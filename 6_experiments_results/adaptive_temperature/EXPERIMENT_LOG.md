# 动态温度蒸馏实验记录

## 实验配置

**日期**: 2026-03-24
**数据**: 8,500条训练样本 (train_full.json)
**验证**: 1,289条样本 (val_fixed.json)
**模型**: unsloth/Qwen3-4B-unsloth-bnb-4bit (4-bit量化)
**方法**: 动态温度软标签蒸馏 (Adaptive Temperature Distillation)

## 超参数

| 参数 | 值 |
|------|-----|
| 学习率 (LR) | 3e-4 |
| Alpha (KL权重) | 0.5 |
| LoRA Rank | 16 |
| Batch Size | 4 (梯度累积4步, 等效16) |
| 序列长度 | 64 |
| Epochs | 3 |
| 温度策略 | v1 (高置信度1.5, 中2.0, 低2.5-3.0) |

## 训练过程

## 对比基线

| 方法 | 准确率 | 中性召回 | 备注 |
|------|--------|----------|------|
| 固定T=2.0 (基线) | 86.50% | 80.5% | 传统软标签蒸馏 |
| **动态温度 (本实验)** | **95.80%** | **94.53%** | 自适应温度缩放 |
| 提升 | **+9.30%** | **+14.03%** | - |

## 关键发现

1. **动态温度有效**: 根据样本置信度自适应调整温度(1.5-3.0)，显著优于固定温度
2. **中性类提升最大**: 从80.5%召回率提升到94.53%，解决了基线最大的短板
3. **模型校准优秀**: ECE仅0.0162，置信度与准确率高度一致
4. **训练稳定**: 3个epoch内Loss稳定下降，无震荡

## 最优配置

通过600条样本快速调参确定:
- **LR**: 3e-4 (对比1e-4, 5e-4)
- **Alpha**: 0.5 (对比0.3, 0.7)
- **温度策略**: v1保守策略 (对比v2激进, v3线性)

## 复现命令

```bash
cd experiments/adaptive_temperature
python3 adaptive_temperature.py \
    --train_data ../../data/curriculum/train_full.json \
    --val_data ../../data/curriculum/val_fixed.json \
    --output_dir ./results/adaptive_temp_full_final \
    --epochs 3 --batch_size 4 --lr 3e-4 --alpha 0.5 \
    --lora_r 16 --full_train
```
